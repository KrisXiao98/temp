链接伪类选择器针对的是超链接类型的标签，即<a>标签。

它主要有4种使用方法：

  • :link 表示未访问的链接
  • :visited 表示已经访问的链接
  • :hover 表示鼠标悬停到上面时
  • :active 表示鼠标点击但是还未松开时
注意： 使用连接伪类选择器时，需要注意使用顺序，编写css时遵循l-v-h-a的顺序。 另外此伪类选择器可以根据不同需求进行使用，可以给指定的a标签进行不一样的操作（比如对某一class进行独有的"lvha"），不一定非得是所有的a标签。
